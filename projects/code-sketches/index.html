Codevember is a challenge for developers to sharpen their creativity and improve their skills. The goal is to build a creative code sketch each day of November using the daily topics from codevember.xyz for inspiration. These are the sketches that I completed for Codevember in 2017 and 2018. I chose to create real-time, animated and interactive 3D sketches with Three.js and tween.js. Because of the time constraint, I've kept each sketch very simple, both visually and conceptually. I modeled the refrigerator and sword in Maya. The rest of the geometry was created procedurally. The sketches can also be found on my Codepen so anyone can easily view and modify the source.
